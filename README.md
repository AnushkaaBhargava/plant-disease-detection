# Smart Agriculture Suite 🌾

A machine learning-powered agriculture platform that helps farmers make data-driven decisions through crop recommendation, fertilizer guidance, weather analysis, and plant disease detection.

## Features

* Crop Recommendation using soil and environmental parameters
* Plant Disease Detection using Machine Learning
* Fertilizer Recommendation
* FastAPI Backend for Predictions
* Trained XGBoost and Decision Tree Models

## Tech Stack

* Python
* FastAPI
* XGBoost
* Decision Tree
* Pandas
* NumPy
* Machine Learning

## Modules

### Crop Recommendation

Recommends suitable crops based on:

* Nitrogen (N)
* Phosphorus (P)
* Potassium (K)
* pH
* Temperature
* Humidity
* Rainfall

### Plant Disease Detection

Predicts plant diseases using trained machine learning models.

### Fertilizer Recommendation

Provides fertilizer suggestions based on crop and soil conditions.

## Project Structure

* `backend/` - Backend services
* `fastAPI/` - API endpoints
* `training/` - Model training code
* `PlantDiseaseApp/` - Disease prediction module
